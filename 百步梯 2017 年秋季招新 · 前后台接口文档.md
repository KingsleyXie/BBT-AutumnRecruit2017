## 百步梯 2017 年秋季招新 · 前后台接口文档

### 报名

```json
POST ./register.php
{
    "name": "value", //姓名
    "gender": "value", //性别
    "college": "value", //学院
    "grade": "value", //年级
    "dorm": "value", //宿舍号
    "tel": "value", //11位手机号
    "department1": "value", //第一志愿
    "department2": "value", //第二志愿（选填）
    "intro": "value" //自我介绍（选填）
}

RESPONSE:
//On Success:
{
    "code": 0, //报名成功
    "errMsg": "success" //默认值
}

//On Failure:
{
    "code": 1,
    "errMsg": "写入数据库失败" //报名失败
}

{
    "code": 233,
    "errMsg": "必填项中含有空值" //数据错误
}

{
    "code": 2333,
    "errMsg": "数据库连接出错，请联系管理员" //配置错误
}
```
